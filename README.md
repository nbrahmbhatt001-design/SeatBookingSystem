# Seat Booking System (Java)

This is a console-based seat reservation system built in Java.

## Features
- Book a seat
- Cancel a reservation
- Move a reservation
- Find N adjacent seats
- Display seating chart
- File input/output support

## How to Run
1. Compile the program:
   javac Main.java

2. Run the program:
   java Main

## Project Structure
- Main.java → Entry point
- Menu.java → Handles user interaction
- SeatingChart.java → Core logic (booking system)
- seating1.txt → Input file for seating layout
